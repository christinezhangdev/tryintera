# Persona images

Add your own images here to scatter across the page like a street of people:

- `person-1.png` through `person-17.png`

Use PNGs with **transparent backgrounds** so they blend naturally with no white boxes. Each image will appear as an individual figure along the left and right edges of the page.

You can also use the "Customize persona images" link in the footer to upload images directly in the browser.
